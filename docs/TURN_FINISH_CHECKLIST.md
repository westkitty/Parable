# Parable Turn Finish Checklist

Generated by `node scripts/parable-turn-guard.mjs finish --summary "..."`.

## Required Finish Behavior

Before ending a work turn:

- Run syntax checks on JavaScript modules and scripts.
- Run `node scripts/verify-parable.mjs`.
- Confirm all required files exist and are non-empty.
- Confirm static serving works locally when the environment allows it.
- Fetch served core files and confirm HTTP 200 responses when the environment allows it.
- Inspect `src/main.js` and runtime chunks for stale or missing references.
- Record any unverifiable WebGL or browser limits honestly.
- Update `docs/GENERATED_ASSET_LIST.md` if asset needs changed.
- Append `Parable_Bible.md` last.
- Package only after verification and docs are in place.

## Required Delta Review

Compare the finish state against `.parable-turn/start-snapshot.json`:

- added files
- changed files
- removed files
- current latest bible entry
- known limitations

## Current Known Browser Limit

The static app imports Three.js from `https://esm.sh/three@0.164.1`. Local syntax and static-file verification do not prove live WebGL execution. A real browser with network access is required unless Three.js is vendored locally.
